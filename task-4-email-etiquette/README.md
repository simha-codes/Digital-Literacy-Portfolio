# Task 4 – Email Etiquette

## Objective

To understand professional email writing and communication etiquette.

## Professional Email

A formal email was created using proper subject line, greeting, and closing.
This helps in professional communication.

## Email Etiquette Rules

* Use clear subject line
* Use formal language
* Keep message short
* Avoid grammatical errors
* Include signature

## Learning Outcome

This task helped me learn professional communication skills.
I understood how to write clear and respectful emails.
